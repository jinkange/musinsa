1. 무신사 계정 셋팅
 - 무신사페이 등록(신용카드 추천 체크카드나 계좌는 인증이 뜰 수 있음)
 - 배송지 미리 입력후 기본 배송지로 저장
 - 무신사페이 비밀번호 6자리 설정

2. data 폴더안에 자신의 크롬 위치를 입력
 - 바탕화면 크롬 우클릭 -> 속성 ->대상을 복사
 - chrome.txt 에 붙혀넣기
 - ※주의 양옆 쌍따옴표는 제거

3. data 폴더에 실행시킬 매크로의 번호를 입력
 - number.txt에 값 넣기
 - ex)1 이라고 저장하면 1번 매크로가되며 폴더를 복사해서 2 등등의 넘버를 순차적으로 넣으면 개별로 작동하는 매크로가 됨 
 - 다계정일시 적용 기본값은 1

4. data 폴더에 무신사페이 비밀번호 6자리 입력
 - password.txt에 비밀번호 6자리 입력

5. musinsa.exe 실행
 - 원하는 상품의 구매 주소 입력
 - 원하는 사이즈 입력(사이즈가 품절이거나 구매할 수 없다면 랜덤 선택)
 - 사이즈 미입력시 랜덤 선택

6. 크롬 실행 후 
 - 크롬 실행 후 건들지 말것
 - 상품 페이지로 이동 후 로그인 화면으로 이동됨
 - 로그인을 진행하면 다시 해당 상품페이지로 이동하면 작동 시작 및 대기 상태

7. 구매완료
 - 상품이 품절 -> 구매가능이 되거나 판매 예정 -> 구매가능이 될떄 작동됨
 - 상품이 자동 결제 되며 무신사 페이까지 자동 입력되며 구입됨
